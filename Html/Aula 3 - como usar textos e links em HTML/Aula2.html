Introdução ao HTML

Definição e Comandos

O HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

O uso de "Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites, pois os links são um aspecto fundamental da web.
Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.


Toda a definição de Hipertexto apresenta uma estrutura bem definida, no qual, utilizamos tags de marcação para indicar os seus atributos e como se relacionam dentro do contexto, cada uma delas apresenta uma < tag de início > e uma < tag de fechamento /> demarcando cada instrução. Além disso, cada tag pode possuir um ou mais atributos, transformando a formatação básica da marcação, adicionando cor, diferentes fontes e até arquivos de mídia como imagens ou vídeos.

Versões do HTML

  • HTML 1.0
    • Versão básica do HTML, possuí suporte para elementos de texto e controle de imagens
    • Não possuí suporte para tabelas ou fontes.
  • HTML 2 
    • Aprimorar a versão 1.0, foi lançado em 1995
    • Incluir suporte a tabelas, botões, textbox.
  • HTML 3.2
    • Lançado em 1997, possuí suporte ao CSS
    • Criado para ser implementado em diversos browsers
  • HTML 4.01
    • Lançado em 1999 - expandindo o suporte ao CSS
    • Possuí upgrade para suportar frame tags 
  • HTML 5 
    • Lançado em 2014, suporta muita das novas tecnologias, audio, senha e email
